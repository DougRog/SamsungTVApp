Images Directory
================

Please add the following images to this directory:

1. logo.png
   - Recommended size: 800x400 pixels or larger
   - Format: PNG with transparent background
   - This will be displayed on the splash screen at 50% scale (configurable)

2. icon.png
   - Size: 512x512 pixels
   - Format: PNG
   - This will be your app icon for Samsung TV

3. placeholder.png (optional)
   - Size: 320x180 pixels
   - Format: PNG
   - Used as fallback when video thumbnails fail to load

Note: The logo scale can be adjusted in app-config.json under splash.logoScale
